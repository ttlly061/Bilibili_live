# bilibili_live

##### 简单的实现 b 站直播互动区功能

```
# 榜单及切换榜单
1. 支持 hover 展开
2. 支持懒加载

# 弹幕区
1. 支持发送弹幕，弹幕列表支持滚动
2. 支持按钮点击清空全部弹幕
```
